X-keys USB HD15-GPIO Setup Utility

1. Plug in one X-keys USB HD15-GPIO.
2. Start HD15GPIOSetupUtility.exe, make sure PIEHid32Net.dll is in the same directory.
3. The status bar on the bottom should read "X-keys USB HD15-GPIO connected". The configurations of each of the configurable pins are set to their current configuration.
4. To change the configuration for a pin select the corresponding radio button. Repeat for all pins needing changed.
5. When done setting all the pins to their desired configuration click Set Configuration.
6. The checkboxes next to the pins can be used to control the output for that pin, checked for on, unchecked for off.
7. Inputs can be tested with the last section of labels on the bottom. When an input is pressed the corresponding label will turn green, when it is released the green will go away.
