X-keys XBE Series GPIO Setup Utility

1. Plug in the XBE series device. Supported devices are XBE-24/XBM-24 Module, XBE-18/XBM-18 Module, XBE-12 Jog Shuttle/XBM-12 Jog Shuttle Module, XBE-12 Handwheel/XBM-12 Handwheel Module, XBE-12 Trackball/XBM-12 Trackball Module, and the XBE-14 T-bar/XBM-14 T-bar Module.
2. Start XBEGPIOSetupUtility.exe, make sure PIEHid32Net.dll is in the same directory.
3. From the drop down box, select desired XBE device if there is more than one connected.
4. The status bar on the bottom should indicate the name of the connected device and that it is currenlty selected. The current configurations of each of the GPIO pins are shown in the corresponding groupbox.
5. To change the configuration for a pin select the desired setting radio button. Repeat for all pins needing to be changed.
6. When done setting all the pins to their desired configuration click Set Configuration.
7. Once configuration and hardware connections to the pins are complete, they can be tested.
7. For output pins use the Control Output section. The checkboxes next to the pins can be used to control the output for that pin, checked for on, unchecked for off.
8. For inputs pins use the Test Inputs sectionn. When an input is switched on the corresponding label will turn green, when it is switched off the green will go away.
